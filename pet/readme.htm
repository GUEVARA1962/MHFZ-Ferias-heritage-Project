○基礎知識
・大闘技会は夜間のみ
・1vs1か2vs2で相手モンスターを倒す
・ペットは闘技演習「陸の女王と対峙せよ！」までクリアが条件
・途中、力尽きた場合10秒間動けない
・大闘技会中に使用したアイテムは終了後に戻ってくる

○モンスターのステータス
・体力は闘技LV（強さ＋調子）により変化
・友好度は、笛による指示が成功した時の逆鱗ゲージの上昇に影響

○逆鱗ゲージとは
・逆鱗笛での攻撃時間の長さ。基本は15秒で最大105秒まで増加
